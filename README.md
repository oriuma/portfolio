## Customization

To personalize this portfolio:

1. Update the profile information in `index.html`
2. Add your own projects and descriptions
3. Update social media links
4. Modify color scheme in CSS variables (`:root` section in `style.css`)
5. Add your own images if needed

## Color Scheme

The site uses a professional blue and dark theme:
- Primary: #2563eb (Blue)
- Background: #0f172a (Dark Blue)
- Surface: #1e293b (Slate)
